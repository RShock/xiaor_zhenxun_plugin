# 小R的zhenxun插件仓库

对每个插件进行了不计其数的细节优化且适配当前最新的真寻版本。

## 插件列表

### zhenxun_bot_route2 — 负载均衡插件

将消息分配给群内的多个 bot 负责。每日发送消息数量大于配置时拦截发送，有效防止风控的发生。配置文件在文件夹里。推荐每个消息的 limit 在 350 以下。

[下载](https://download-directory.github.io/?url=https://github.com/RShock/xiaor_zhenxun_plugin/tree/master/zhenxun_bot_route2)

### nonebot_plugin_handle — 初代猜成语

猜成语游戏，进行了大量优化，包括抢答等功能。

[下载](https://download-directory.github.io/?url=https://github.com/RShock/xiaor_zhenxun_plugin/tree/master/nonebot_plugin_handle)

### nonebot_plugin_handle2 — 二代猜成语

在 1 代玩家作弊太多后原创的防作弊版本。

[下载](https://download-directory.github.io/?url=https://github.com/RShock/xiaor_zhenxun_plugin/tree/master/nonebot_plugin_handle2)

### nonebot_plugin_steam_info — Steam 状态插件

优化了头像显示和离线提示的 Steam 状态查询插件。

[下载](https://download-directory.github.io/?url=https://github.com/RShock/xiaor_zhenxun_plugin/tree/master/nonebot_plugin_steam_info)
