# handle - 猜成语

## 游戏规则
- 发送"猜成语"开始默认精确模式
- 支持模糊模式、简单模式、简单模糊模式
- 每人每天最多发起3次游戏，开局创建记录用于次数统计

## 流程注意
- 正常胜负、抢答胜负、主动结束、超时都必须先保存 `is_win/attempts`，再清理内存状态
- 该插件为 handle2 的流程来源；修复统计流程时需保持两个插件一致

## 文件结构
- `__init__.py` - nonebot插件入口，命令注册与游戏流程
- `data_source.py` - 游戏核心逻辑与绘图
- `config.py` - 插件配置
- `models.py` - 数据库模型（HandleRecord）
- `utils.py` - 拼音、成语、字体等工具函数
- `simulate_handou.py` - 模拟测试脚本
